> 使用说明：
```
node > 8.6.0
yarn > 1.6.0
```
> 安装 

```

yarn global add starter-kit-cli

npm install starter-kit-cli -g

```
> 使用方法

#### 创建一个模板:
```
starter-kit-init <文件夹> [模板类型: pc / mobile] 

如: starter-kit-init test pc
```
#### 删除当前模板:
```
starter-kit-del

? 请输入要删除的文件目录名称 <输入你要删除的模板目录名>

? 确定要删除吗？(y/n)  回车即可(或者输入y)

```
